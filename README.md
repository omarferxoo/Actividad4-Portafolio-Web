## Portafolio personal

Este proyecto presenta un portafolio personal desarrollado con HTML, Bootstrap,
CSS personalizado y JavaScript. El objetivo es mostrar informacion profesional
del estudiante, habilidades, proyectos realizados y datos de contacto.

## Framework CSS y plantilla utilizada

Se utilizo **Bootstrap 5.3.3 por CDN** como base de estilos. El portafolio esta
basado en la plantilla **Freelancer** de Start Bootstrap, adaptada a un perfil
academico y profesional.

Link de Bootstrap:

https://getbootstrap.com/

Link de la plantilla:

https://startbootstrap.com/theme/freelancer

La personalizacion visual se agrego en el archivo `css/portafolio.css`.

## Secciones del portafolio

- **Inicio:** presenta el nombre del estudiante, una descripcion breve y la foto
  de perfil.
- **Sobre mi:** incluye una descripcion del perfil academico y profesional.
- **Habilidades:** muestra skills relacionadas con desarrollo web y publicacion
  de proyectos.
- **Proyectos:** presenta trabajos o practicas realizadas.
- **Contacto:** contiene datos basicos para contactar al estudiante.

## Proceso de creacion

1. Se creo la estructura principal en `index.html`.
2. Se agrego Bootstrap por CDN para usar una plantilla responsiva.
3. Se organizaron las secciones solicitadas: inicio, sobre mi, habilidades,
   proyectos y contacto.
4. Se creo el archivo `css/portafolio.css` para modificar colores, espaciado,
   tarjetas, hero, foto de perfil y vista movil.
5. Se creo el archivo `js/portafolio.js` para cerrar el menu en dispositivos
   moviles al seleccionar una seccion.
6. Se preparo la carpeta `img` para agregar manualmente la foto de perfil,
   capturas y logotipo de portada.
